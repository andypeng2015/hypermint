# hypermint

hypermint = wasm + tendermint

This is a very experimental project.

## Build

```
$ dep ensure
$ make build
```

## Getting started

First, you need to initialize genesis state.

```
$ make init
```

Next, run a blockchain node:

```
$ make start
```
